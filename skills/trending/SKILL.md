# Trending Skill

Track trending topics and hot searches.

## Usage

### Check Trending (Various Platforms)

```bash
# Weibo Hot Search
trending weibo

# Zhihu Hot
trending zhihu

# Bilibili Popular
trending bilibili

# Douyin Trending
trending douyin

# GitHub Trending
trending github

# WeChat Hot Articles
trending wechat
```

### Common Commands

| Platform | Command | Description |
|----------|---------|-------------|
| 微博 | trending weibo | 微博热搜 |
| 知乎 | trending zhihu | 知乎热榜 |
| Bilibili | trending bilibili | B站热门 |
| 抖音 | trending douyin | 抖音热榜 |
| GitHub | trending github | GitHub趋势 |
| 微信 | trending wechat | 微信热文 |
