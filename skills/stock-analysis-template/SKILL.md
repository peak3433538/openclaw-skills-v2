# 股票深度分析模板

## 标准化股票分析报告模板

### 一、基本信息
- 股票代码
- 股票名称
- 上市板块
- 主营业务

### 二、今日行情数据
| 项目 | 数据 |
|---|---|
| 最新价格 | xxx元 |
| 涨跌幅 | xx.xx% |
| 涨跌额 | xx元 |
| 总市值 | xx亿元 |
| 流通市值 | xx亿元 |
| 市盈率(TTM) | xx倍 |
| 市净率 | xx倍 |
| 60日涨跌幅 | xx% |
| 年初至今 | xx% |

### 三、公司竞争力分析
- 核心业务介绍
- 商业模式（B2B2C/平台型等）
- 核心竞争力（技术/渠道/资源壁垒）
- 产业链位置

### 四、产业链分析
- 上游分析
- 下游分析
- 公司位置
- 议价能力

### 五、财务表现
- 营收与增长
- 盈利能力（毛利率、净利率）
- 现金流
- 研发投入
- 财务特点

### 六、所属板块及热点
- 所属行业
- 概念板块
- 今日板块热点

### 七、10日主力资金情况
| 时间周期 | 主力资金流向 |
|---|---|
| 近10日 | xxx万元 |
| 近5日 | xxx万元 |
| 今日 | xxx万元 |

### 八、各大机构评论热点
- 机构观点
- 市场热点

### 九、政策影响分析
- 利好政策
- 政策影响

### 十、基金持仓情况
| 项目 | 数据 |
|---|---|
| 持有基金数量 | xx只 |
| 较上期变化 | 增加/减少x只 |
| 主要持仓基金 | xxx |
| 持仓比例 | xx% |

### 十一、股东情况分析
| 项目 | 数据 |
|---|---|
| 股东户数 | x.xx万户 |
| 较上期变化 | xx% |
| 十大流通股东占比 | xx% |

### 十二、热点信息追踪
- 行业动态
- 公司新签客户
- 技术进展
- 行业竞争

### 十三、风险提示
1. 风险1
2. 风险2
3. 风险3
...

### 十四、投资建议
- 短期建议
- 中期建议
- 长期建议
- 操作策略

---

## 使用方法

当用户要求分析股票时：
1. 获取股票基本信息
2. 获取今日行情数据
3. 分析所属板块热点
4. 分析主力资金
5. 分析政策影响
6. 分析基金持仓
7. 分析股东情况
8. 整理投资建议

输出格式：Word文档，通过邮件发送
