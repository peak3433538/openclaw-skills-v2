# RSS Reader Skill

Read and manage RSS feeds.

## Setup

This skill provides RSS feed reading capabilities.

## Usage

### Add RSS Feed
```bash
# Add a feed to the reader
rss add <feed_url>
```

### List Feeds
```bash
# List all subscribed feeds
rss list
```

### Read Feed
```bash
# Read latest items from a feed
rss read <feed_name>
```
